# Convert Elapsed Time to Date

Convert Elapsed Time to Date

## Usage

``` r
offset_time_as_date(t, start_date)
```

## Arguments

- t:

  time in decimal years since start_date

- start_date:

  start date in decimal years

## Examples

``` r
if (FALSE)  offset_time_as_date(2.109589, 2007)  # \dontrun{}
```
